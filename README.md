# Vivi

Personal productivity dashboard. Three modules: Home (Villa Vista + Amy Lane todo pads and a day agenda), monthly Calendar with repeating events, and Projects with a todo pad per project.

## Install on her computer
1. Serve the folder: `python3 -m http.server 8080` inside this folder, or host it (GitHub Pages works).
2. Open it in Chrome or Edge.
3. Click the install icon in the address bar. Vivi opens as its own app.

## Install on her phone
Host it online first (GitHub Pages). Open the link in Safari or Chrome, then "Add to Home Screen".

## Data
Everything saves in the browser (localStorage) on that device. No account needed. Use the export option for backups (Store.exportJSON in console for now).
